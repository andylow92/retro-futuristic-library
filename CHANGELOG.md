# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2024-11-15

### Added
- Initial release of Retro Glass UI
- `GlassButton` component with multiple sizes and variants
- `GlassCard` component with title, subtitle, and effects
- `GlassPanel` component with header/footer support
- `GlassBadge` component with pulse animation
- `GlassInput` component with theme support
- `GlassContainer` component for layouts
- Full TypeScript support
- Six color themes: pink, orange, cyan, purple, neon, blue, gradient
- Four glass variants: light, medium, heavy, neon
- Optional effects: scanlines, dots, neon borders, glow
- Comprehensive documentation
- Example usage file

### Features
- Retro-futuristic design aesthetic
- Liquid glass glassmorphism effects
- Static retro grid backgrounds
- Comic book inspired styling
- Fully typed with TypeScript
- Tree-shakeable exports
- Zero dependencies (peer deps: React only)
- Customizable themes and variants
